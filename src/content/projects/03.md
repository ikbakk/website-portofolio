The tradeoff: keep the product experience familiar while improving the
long-term maintainability of the mobile codebase.

I maintained and enhanced a Flutter-based short-form drama streaming app,
including feature work, REST API integration, multilingual localization, and
production issue fixes. I also migrated a subscription ebook application from
React Native to Flutter and built mobile interfaces connected to Sanity CMS
for dynamic content management.
