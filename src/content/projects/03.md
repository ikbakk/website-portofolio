title: 'CV Builder - local-first PDF resume editor'
---

The constraint that shaped it: give users control over their resume templates
and personal data without making PDF generation feel technical.

I built a React editor that stores structured resume data locally, transforms
it into Typst source, and renders the resulting PDF for preview. The project
keeps UI, application services, domain types, templates, and persistence
separate so the editor remains understandable as templates evolve.
