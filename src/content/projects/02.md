title: "Dense internal workflow tool for ops teams"
---

The constraint that shaped it: tables with 50+ columns, 100ms column
resize, and a save model that could not lose work to a network blip.

Wrote a draft + commit layer that turned the page into a stable editing
surface, and a query layer that survived bad data without throwing. The
team got back roughly an hour per ops shift once the workspace replaced
the spreadsheet chain.
