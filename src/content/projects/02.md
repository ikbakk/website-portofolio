title: "Dense internal workflow tool for ops teams"
---

An internal tool for a 30-person ops team processing thousands of records a
day. The constraint was information density and keyboard speed, not visual
delight. Built a small component kit, a keyboard layer over react-aria,
and a query system that survived bad data.

The constraint that shaped it: tables with 50+ columns, 100ms column
resize, and a save model that couldn't lose work to a network blip. Wrote
a draft + commit layer that turned the page into a stable editing surface.
