title: "Real-time IoT monitoring dashboards"
---

The constraint that shaped it: telemetry changes constantly, but the
interface still has to feel readable and stable for people monitoring
connected devices and operational metrics.

I built React dashboards with MQTT-based data streams, Firebase integration,
React Query for asynchronous fetching and client-side caching, and Recharts
for operational visualizations. The work turned raw device data into views
that users could scan quickly and trust during day-to-day monitoring.
